# Perl Script for Entitlement Check

## Usage: all_in_one.pl [you base dir]
NOTE: dir should not end with '/'

For Example:
c:>all_in_one.pl c:/workspace/esp

## Requirement:
1.You need perl installed. Either ActivePerl or StrawberryPerl
2.You need below cpan module installed:
	a.File::Find
	b.Excel::Writer::XLSX
3.Put below perl script in any folder.
4.Run all_in_one.pl.

## Note:
After execution, there are 2 .txt files and one .xlsx file.

Have Fun!
